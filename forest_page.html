โครงการป่าชุมชนบ้านพุเม่น

รายละเอียด
ชื่อป่าชุมชน: โครงการป่าชุมชนบ้านพุเม่่น
ปีทีเริ่มโครงการ: พ.ศ. 2562
เนื้อที่ : 344 ไร่ 0 งาน 0 ตารางวา
ประเภทป่า : ป่าตาม พ.ร.บ. ป่าไม้ พ.ศ.2484
สถาพทั่วไป : พื้นที่เป็นภูเขาลูกโดด เนินเขา และชายเขา มีความสูงไม่มากนัก บริเวณโดยรอบเป็นพื้นที่ทำการเกษตรไร่มันสำปะหลัง และข้าวโพดเลี้ยงสัตว์ ลักษณะดินเป็นดินร่วนปนทราย
หมู่บ้าน : พุเม่น (หมู่ 3)

ปีทีเริ่มโครงการ: พ.ศ. 2562
เนื้อที่ : 147 ไร่ 0 งาน 0 ตารางวา
ประเภทป่า : ป่าตาม พ.ร.บ. ป่าไม้ พ.ศ.2484

หมู่บ้าน : ไตรคีรี (หมู่ 6)
